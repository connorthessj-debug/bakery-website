

# Wireframes & UI Layout вҖ” "Le Besian Balls" Bakery Website

## Complete Design System & Wireframe Documentation

---

## 1. DESIGN PRINCIPLES

Before diving into layouts, I'm establishing the guiding principles that will anchor every design decision:

1. **Mobile-First, Always** вҖ” Every layout starts at 375px and scales up
2. **Accessibility-First** вҖ” WCAG 2.1 AA minimum; every interaction has a keyboard and screen-reader path
3. **Appetite Over Overwhelm** вҖ” The site should make you *hungry*. Big, beautiful imagery. Warm tones. Breathing room.
4. **Frictionless Ordering** вҖ” From homepage to cart in вүӨ 3 clicks
5. **Content-Owner Friendly** вҖ” Layouts designed for Sanity CMS content blocks so the bakery owner can update without a developer
